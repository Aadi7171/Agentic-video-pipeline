# Agentic AI Video Pipeline 🎬🤖

An autonomous, multi-agent AI video production engine. This project takes a simple text prompt and transforms it into a platform-ready video. It orchestrates a chain of specialized AI agents built on top of **Google Gemini 2.5** and the **VideoDB** programmable video platform.

---

### 🧠 **Architecture & Multi-Agent Flow**
This is not a black-box wrapper. The pipeline uses a custom orchestration loop where dedicated agents act sequentially:

1. **🖋️ The Scriptwriter Agent (Gemini):**
   * Semantically breaks down the user’s prompt into a JSON-structured script.
   * Generates localized dialogue and precise visual cues/keywords for the next layers.

2. **🎙️ The Voice Agent (TTS to VideoDB):**
   * Concatenates the generated dialogue.
   * Uses `gTTS` to synthesize a natural-sounding voiceover.
   * Securely uploads the audio directly into the VideoDB asset store.

3. **🎞️ The Asset Agent (Pexels Integration):**
   * Acts as an AI Researcher mapping Gemini's visual keywords to real-world stock footage using the Pexels REST API.
   * Immediately ingests the high-definition MP4 URLs via VideoDB’s server-to-server sync.

4. **✂️ The Editor Agent (VideoDB Timeline SDK):**
   * The "Director" of the pipeline.
   * Uses the programmable VideoDB `Timeline` to chronologically slice and stitch the B-roll assets.
   * Overlays the voiceover audio track across the final combined footage.
   * Compiles the manifest and exports a final, edge-cached streaming URL.

### 🚀 **Installation & Setup**

#### **1. Clone the repository**
```bash
git clone https://github.com/your-username/agentic-video-pipeline.git
cd agentic-video-pipeline
```

#### **2. Install Dependencies**
We recommend using a Python virtual environment.
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

#### **3. API Configuration**
Rename the `.env.example` file to `.env` and configure your keys:
* Get your VideoDB API Key from [console.videodb.io](https://console.videodb.io)
* Get your Google Gemini API Key from [AI Studio](https://aistudio.google.com)
* Get your Pexels API Key from [Pexels Developer](https://www.pexels.com/api/)

#### **4. Run the Pipeline**
```bash
python src/main.py
```
* **Input:** `Create a short video about the history of Artificial Intelligence.`
* **Output:** A playable URL generated by VideoDB containing your synthesized video.
